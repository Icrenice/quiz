US04 Antwoord geven op een vraag

  • Het is voor de speler mogelijk precies één antwoord te selecteren

US05 Is het antwoord correct?
  • De quiz registreert of het gegeven antwoord juist of onjuist is.
  • Als de speler het juiste antwoord heeft gegeven moet de speler positieve feedback krijgen.
  • Als de speler een onjuist antwoord geeft wordt de speler gewezen op de fout.
  • Het juiste antwoord wordt getoond.

  • Gegevensvalidatie

US11 Opslaan score via Quiz-API
  • Als alle vragen zijn beantwoord zijn is het studentnummer en de score van de speler verstuurd naar de API
  • Indien de API een foutmelding geeft wordt deze foutmelding in de quiz getoond op een manier zodat de quizmaster kan zien wat er mis is gegaan. De gebruiker ziet dat het opslaan van de scores niet is gelukt.
